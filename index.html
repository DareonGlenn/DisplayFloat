Blocks Elements

Block elements start each elemnts on a new line and take up the entire width of the page. Examples are : div ,h1,p,ul,ol,section ,article,footer and header.

Inline elements

Inline elemnts do not start on a new line and only take up as much width as they need to. Examples are: a, img, and span.
